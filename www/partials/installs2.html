• Google Chrome

    Google Chrome is the most popular browser for web developers because of the powerful tools it provides to test code, manipulate documents, and measure performance.

  • Google Cast extension

    Once Chrome is installed, open it and visit the Chrome Store page for Google Cast and install it. This will allow you to share your browser screen in the classroom.

  • Sublime Text

    1. Visit the Sublime Text website and click the link that corresponds to your operating system. Then install once the download is complete.

    2. Install the Sublime Text package control which, in turn, allows you to easily install and manage tons of really cool add-ons for customizing your Sublime editor.

    3. Read this excellent introduction to getting the very most out of Sublime Text. These tips will make coding more enjoyable and increase your speed.
